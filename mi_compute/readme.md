## 目前完成的功能：
        1、读入一个栅格地图，并对其处理并展示。
        2、在栅格地图上运行raycast算法，将一条激光束上的格子挑出来。
        3、计算一条激光束上的互信息，
        假设激光束（0，0）到（5,4） （5,4）为占据，其余全部为free
